Part 1: The Rules

A and An are indefinite articles. We use them with singular, countable nouns when we are talking about something general or non-specific.

When to use A vs. An

  • Use A before a word that begins with a consonant sound.
    Example: a cat, a dog, a university (university starts with a 'y' sound)
  • Use An before a word that begins with a vowel sound (a, e, i, o, u).
    Example: an apple, an egg, an hour (hour starts with an 'o' sound)

When to use Some

Some is a quantifier. We use it with plural countable nouns and uncountable nouns when the amount is not specific or is unknown.

  • Use Some with plural countable nouns.
    Example: some books, some cars, some friends
  • Use Some with uncountable nouns (things you cannot count, like liquids, concepts, materials).
    Example: some water, some information, some rice
